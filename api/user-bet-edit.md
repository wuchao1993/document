#API-用户修改竞猜接口

**简要描述：** 

- 用户修改竞猜接口

**请求URL：** 
- 正式接口： `https://hello-coder.com/api/wxapp/User/editBet`
- 测试地址： `https://hello-coder.com/api/wxapp/User/editBet` 
  
**请求方式：**
- POST

**参数：** 

 - 接口参数：

|参数名|必选|类型|说明|
|:----    |:---|:----- |----- |
| id | Y | int | 修改id |
| bet | Y | int | 1涨2跌 |
| betId | Y | int | 盘口id |

 **返回示例**

```json
{"code":1,"msg":"竞猜成功"}
```
